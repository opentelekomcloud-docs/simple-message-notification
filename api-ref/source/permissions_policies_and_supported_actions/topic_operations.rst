Topic Operations

| Permission       | API                                                      | Action           | IAM Project | Enterprise Project |

+------------------+----------------------------------------------------------+------------------+-------------+--------------------+
| Querying a topic | GET /v2/{project_id}/notifications/topics                | smn:topic:list   | Y           | Y                  |
